Frequently Asked Questions

Everything you need to know about getting started on CLEC.live

What is the difference between an 'Event Creator' and an 'Event Organizer'?

Event Creator is a platform role — anyone who creates an Event Concept on CLEC.live becomes the Event Creator for that event. This could be an Artist, Venue owner, or Contributor.

Event Organizer is an occupation/profession — a type of Contributor Profile for professional event organizers (like wedding planners, festival coordinators, etc.). Event Organizers register as Contributors on the platform.

Can I create events without registering?

No, you need at least one collaborator Profile (Artist, Venue, or Contributor) to create events. Audience Profiles cannot create events.

I have an Audience Profile. Can I create events?

No, Audience Profiles are for browsing and attending events. To create events, you need to add an Artist, a Venue, or a Contributor Profile to your account. You can do this from the My Profiles page.

Can I have multiple profile types?

Yes! You can create up to 4 different Profiles (Artist, Venue, Contributor, Audience) under one account. This is useful if you play multiple roles in the live events community — for example, you might be both a musician (Artist) and a sound engineer (Contributor).

What profile type should I choose if my role isn't listed?

When registering, select "Other (please specify)" from the occupation dropdown. You can then:

  • Describe your role in your own words (up to 100 characters)
  • Choose which Profile type best fits: Artist (musicians, performers), Venue (event spaces), or Contributor (services: sound, lighting, catering, photography, etc.)

How do I switch between my Profiles?

Use the profile switcher in the top navigation bar. Your active Profile determines which events and opportunities you see, and which actions you can take. You can switch Profiles anytime from the My Profiles page.

What's the difference between 'Create My Event' and 'Collaborate on Events'?

Create My Event: Start your own event from scratch or use a template. You become the Event Creator and manage all aspects of the event, including inviting collaborators.

Collaborate on Events: Find existing events looking for Artists, Venues, or Contributors. Browse open Slots and apply to join events created by others.

Can I browse events without creating a Profile?

Yes! You can browse published events without an account. However, to apply for collaboration opportunities, create events, or save favorites, you'll need to register and create at least one Profile. Start by clicking Join CLEC.live.
